# 예시 라이브러리 (Few-shot Learning)

이 폴더에는 AI가 참고할 우수한 상세페이지 예시를 저장합니다.

## 📂 폴더 구조

- `minimal/` - 미니멀/클린 스타일 예시
- `rich/` - 풍부한/친근한 스타일 예시

## 📝 이미지 추가 방법

1. 좋은 상세페이지 스크린샷 저장 (PNG 또는 JPG)
2. 적절한 폴더에 저장 (`minimal` 또는 `rich`)
3. 파일명 형식: `example_카테고리_번호.jpg`
   - 예: `example_electronics_1.jpg`
   - 예: `example_fashion_2.jpg`

## ✅ 좋은 예시 기준

- 해상도: 최소 800px 너비
- 완성도 높은 상업용 페이지
- 명확한 섹션 구조
- 가독성 좋은 텍스트
- 적절한 여백

## 🚀 사용법

이미지를 추가하면 자동으로 AI 생성 시 참고됩니다.
별도 설정 불필요!
